# Glam Heaven – Beauty Parlour Booking Management System

A responsive web application developed for managing beauty parlour appointments, service packages, and customer interactions.

## Features
- Appointment booking system
- Service management
- Customer record management
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Project Objective
The project was developed to simplify salon booking workflows and improve customer experience through an organized digital platform.
